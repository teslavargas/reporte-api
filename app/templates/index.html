Bienvenido a la API de Reporte Minoritario

Esta API te permite obtener datos normalizados de los episodios del Reporte Minoritario. Utiliza técnicas de scraping y procesamiento para extraer la información de la web y presentarla de manera estructurada.

Documentación Interactiva (Swagger UI) Documentación Extendida (ReDoc)

Para consultar un episodio, utiliza la siguiente ruta: /data/full/normalized/episode/<número>
